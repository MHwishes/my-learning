1. 启动容器：

   ``` bash
   docker-compose up 
   ```

2. 停掉容器

   ```sh
   docker-compose down 或 docker-compose down --volumes（可以删除volumn 中的缓存）
   ```

3. 删除已经死的容器

   ```sh
   docker rm 容器id
   ```

4. 进入容器

   ```sh
   docker exec -it 容器id /bin/bash
   ```

   

5. 查看活着的容器

   ```
   docker ps
   ```
